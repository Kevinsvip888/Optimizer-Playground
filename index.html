reference

Optimizer reference

SGD
x ← x − α·∇f(x)

Vanilla gradient descent. Simple and interpretable, but slow on ill-conditioned problems and sensitive to learning rate choice.

Momentum
v ← βv + α·∇f(x)   x ← x − v

Accumulates a velocity vector to dampen oscillations and accelerate in consistent gradient directions. Think of a ball rolling down a hill.

Nesterov
∇f(x − β·v)   (lookahead)

Evaluates the gradient at the anticipated future position, giving a "look before you leap" correction that improves convergence.

AdaGrad
G ← G + g²   x ← x − α·g/√G

Adapts the learning rate per parameter by accumulating squared gradients. Excellent for sparse data; can stall on dense problems.

RMSProp
E ← ρE + (1−ρ)g²   x ← x − α·g/√E

Fixes AdaGrad's vanishing learning rate by using an exponential moving average of squared gradients instead of a running sum.

Adam
m̂ = m/(1−β₁ᵗ)   v̂ = v/(1−β₂ᵗ)

Combines Momentum and RMSProp with bias correction for both moment estimates. The most widely used optimizer in deep learning.

AdamW
x ← x(1−αλ) − α·m̂/√v̂

Adam with weight decay applied directly to parameters rather than the gradient — the correct way to do L2 regularization with adaptive optimizers.
